### Setup .env file

```bash
MONGO_URI=your_mongo_uri
PORT=process.env.PORT || 8080
JWT_SECRET=your_secret_key
NODE_ENV=development

EMAIL_USER_NAME=Your_Name
EMAIL_USER=Your_Email_ID
APP_PASSWORD=Email_Password
```

### Start the app

```shell
npm run start
```
